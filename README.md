# ChainDigger
## これなに
いわゆるシャベルで掘れるブロックの一括採掘機能を追加する。Spigot用プラグインです。

## (動作|作成)環境
- Spigot 1.17.1
- Java 16
- Kotlin 1.5.30

## How To Build
以下のコマンドを入力すると、`build/libs/`以下にjarファイルができあがります。

```bash
$ ./gradlew build
```

## 使い方
スニーク状態で一括採掘機能が有効になります。各種シャベルで以下のものが一括採掘可能です。

- 砂利
- 粘土
